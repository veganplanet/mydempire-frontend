The Codex of MydEmpire

This Codex defines how the empire truly works. Every asset, every rule, and every system is part of a connected production network. Understanding this page means understanding how to build a stronger empire.

Core Gameplay Loop

Pack → Land + Blueprint + EMP → Build Factory → Generate EP → Maintain → Upgrade → Earn Share → Expand Empire

The goal is not collecting assets, but converting them into a working industrial system.

Land Rules

Slots

  • L1 Frontier → 1 slot
  • L2 Estate → 2 slots
  • L3 Industrial Zone → 3 slots

B5 Limitation Rule

Each land can have maximum 1 B5 factory.

Upgrade Behavior

Upgrade capability depends on how the factory started:

  • L1:
    • Direct B4/B5 build allowed
    • If started B1/B2 → max upgrade B3
  • L2:
    • Max 1 B5
    • If one B5 exists → other slot max B4
    • If both start low → both can go up to B4
  • L3:
    • If one B5 → other 2 max B4
    • If all start low → 1 can reach B5, others B4

Blueprints

Blueprints define what industry can be built and are consumed during construction.

Levels

  • B1 → Basic
  • B2 → Improved
  • B3 → Advanced
  • B4 → High
  • B5 → Elite

Extra Blueprint Mint

Allowed only when total blueprint count is lower than available slots. EMP spent goes 100% to treasury.

Factories

Factories are built using land + blueprint + EMP.

States

  • Building
  • Active
  • Upgrading
  • Inactive

Build Time

  • B1 → 1 day
  • B2 → 2 days
  • B3 → 3 days
  • B4 → 4 days
  • B5 → 5 days

EMP

EMP is the core utility resource used for:

  • Building factories
  • Upgrades
  • Maintenance
  • Blueprint minting

EMP from packs is soulbound and intended for gameplay use.

Relics

  • +3% EP per relic
  • Max 2 counted
  • Total max boost = 6%

Special Ability

Relic holders can instantly upgrade factories by paying 25% extra cost after required active days are completed.

Writ

Writ is a premium Genesis asset linked to Founder-level strategic value. It represents special privilege within the ecosystem.

EP System

EP (Empire Power) represents total production strength.

  • Generated by active factories
  • Boosted by relics
  • Defines reward share

Production accumulates for maximum 7 days.

Maintenance

  • Cycle: 7 days
  • Cost: 5% of daily EP (rounded, no decimals)
  • Stops production if unpaid
  • No penalty beyond inactivity

Upgrades

Cost

  • B1 → B2: 50 EMP
  • B2 → B3: 100 EMP
  • B3 → B4: 200 EMP
  • B4 → B5: 400 EMP

Active Days Required

  • B1 → B2: 30 days
  • B2 → B3: 60 days
  • B3 → B4: 120 days
  • B4 → B5: 240 days

Rewards

Rewards are distributed based on EP share:

Player Reward = (Player EP / Total EP) × Reward Pool
  • Max emission: 0.5% of treasury
  • Dynamic based on treasury health
  • No guaranteed returns

Shop

  • Buy sealed packs
  • Buy EMP (100 EMP = 1 HIVE)
  • Mint blueprint if eligible

Marketplace

  • Tradable: EMP, Blueprint, Relic, Writ
  • Land trading in future phase
  • EMP fee: 3%
  • Other assets: 5%
  • Trades in HIVE

EMP Rules

  • Minimum price: 75% floor
  • Must list in multiples of 100

P2P Transfer

  • No platform fee
  • Direct transfer allowed
  • Platform not responsible for off-market deals

New Player Guide

  • Buy a Genesis Pack
  • Open and review assets
  • Build first factory
  • Maintain production
  • Upgrade strategically
  • Expand empire
